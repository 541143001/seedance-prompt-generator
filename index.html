📖使用说明
使用流程：
1. 输入你的 DeepSeek API Key（获取地址）
2. 可选：设置视频参数（画幅、时长、清晰度）
3. 输入你的剧本或创意想法
4. 可选：输入优化方向（如"更紧张""加快节奏"），或直接点击优化
5. 点击「优化提示词」，AI 会保留你的所有细节并增强为专业提示词

AI 优化会做什么：
• 保留你剧本中的所有细节和戏剧元素，不会缩减
• 根据视频参数自动适配画幅、时长和清晰度
• 补充镜头运动、景别角度、光线方向和结束姿态
• 添加专业约束（保持单一场景、主体一致、动作连续）
• 输出一段流畅的叙事段落，可直接复制到即梦平台使用

优化后可以：
• 点击「查看对比」查看优化前后的差异
• 点击「撤销」恢复原始内容
• 点击「再次优化」在优化结果基础上继续优化

示例输入：
• 赛博朋克城市夜景，一个穿风衣的男人在雨中走过霓虹灯街道，15秒，横屏
• 一只白虎被绑在树上，画外音A不耐烦地说"你到底咬不咬"，画外音B突然说"绳结松了"
• 产品广告：一款银色手表的特写展示，高端商务风格，旋转展示，10秒
